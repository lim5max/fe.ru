Партнерство

Создание партнерского сайта

При вступлении в партнерскую программу партнеру создается отдельный сайт. После активации партнерский сайт сразу же начнет работу по временному адресу и сможет принимать клиентов для регистрации.

Начало работы

Для начала работы достаточно только активировать партнерскую программу и предлагать ссылку на ваш сайт потенциальным клиентам.

Важные ограничения

  • Все виды спама запрещены
  • Запрещается рекламировать сайт незаконными способами

Настройка и персонализация

Домен и дизайн

Партнер может:

  • Подключить свой домен к партнерскому сайту
  • Полностью изменить дизайн под себя:
    • Дизайн самого сайта
    • Дизайн биллинговой части (личного кабинета)

Управление клиентами

Для каждого клиента партнера создается личный кабинет, где он может управлять всеми услугами.

Отслеживание финансов

Партнер через "историю платежей" может отслеживать все финансовые транзакции своих клиентов.

Просмотр клиентов

У партнера есть возможность просматривать список клиентов.

Тарифные планы

Названия тарифов для клиентов партнера могут отличаться. Мы предлагаем на выбор партнеру несколько тарифных линеек по хостингу.
Для настройки тарифов: обратитесь в службу поддержки.
